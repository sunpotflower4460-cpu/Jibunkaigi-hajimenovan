# Phase 3F: Legal Policy Update

## 目的

Gemini API / Firebaseクラウド保存 / データ削除導線を踏まえて、アプリ内の利用規約とプライバシーポリシーを現在の実装に合わせる。

## 方針

- 端末内保存だけでなく、Firebaseクラウド保存の可能性を明記する
- Gemini APIへ入力テキストや会話文脈が送信される可能性を明記する
- APIキーはブラウザ側ではなくサーバー側で管理されることを明記する
- 端末内データ削除とクラウドデータ削除が別操作であることを明記する
- 広告・課金は初期版では未導入と明記する
- 医療・診断・治療・緊急対応ではないことを再確認する

## 変更内容

- `src/data/legal.ts`
  - 利用規約の最終更新日を更新
  - AI応答に関する条項を追加
  - Gemini API / Firebase / Vercel / Cloudflare に関する説明を追加
  - データ削除条項を追加
  - 広告・課金が未導入であることを明記
  - プライバシーポリシーを現在の機能に合わせて更新

## 手動確認

- 初回画面から利用規約を読める
- 初回画面からプライバシーポリシーを読める
- 設定画面から利用規約/プライバシーポリシー/注意事項を確認できる
- Gemini APIへの送信可能性が明記されている
- Firebaseクラウド保存の可能性が明記されている
- 端末内削除とクラウド削除が別操作であることが明記されている

## 注意

この文面は申請前の実装整合用ドラフトです。法的な最終判断が必要な場合は専門家確認を行う。
