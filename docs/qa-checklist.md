# じぶん会議 はじめの版 QAチェックリスト

App Store初期版へ進める前に、Vercel Preview・スマホ実機・ブラウザで確認する項目です。

## 1. 起動確認

- [ ] 初回表示でイントロ画面が出る
- [ ] 「会議をはじめる」でホームへ進む
- [ ] リロードしても真っ白画面にならない
- [ ] 端末内保存データが壊れていてもErrorBoundaryで復帰できる

## 2. 入力と会話

- [ ] 空入力では送信できない
- [ ] 1行入力を送信できる
- [ ] 長文入力を送信できる
- [ ] 送信後に入力欄が閉じ、エージェント選択へ進む
- [ ] レイ・ジョー・ケン・ミナ・サトウがそれぞれ応答する
- [ ] 委ねるがランダムに応答する
- [ ] 心の鏡が応答する
- [ ] OTHERSが表示される

## 3. 生成中の操作

- [ ] 応答生成中はセッション切替できない
- [ ] 応答生成中は削除できない
- [ ] 応答生成中はモード変更できない
- [ ] 生成完了後に操作可能へ戻る
- [ ] 応答が別セッションへ誤保存されない

## 4. セッション管理

- [ ] 新しい問いを作れる
- [ ] セッション一覧に表示される
- [ ] セッションタイトルを編集できる
- [ ] ピン留めできる
- [ ] セッション削除できる
- [ ] 現在開いているセッション削除後に安全にホームへ戻る

## 5. メッセージ操作

- [ ] メッセージをタップしてツールバーを表示できる
- [ ] コピーできる
- [ ] メッセージ削除できる
- [ ] OTHERS展開中に削除しても表示が崩れない

## 6. 名前設定

- [ ] 名前変更モーダルを開ける
- [ ] 名前を保存できる
- [ ] 空文字は保存されない
- [ ] 30文字を超えても崩れない

## 7. スマホ表示

- [ ] iPhone幅でヘッダーが崩れない
- [ ] サイドバーを開閉できる
- [ ] 横スクロールのエージェントボタンが押しやすい
- [ ] 入力欄がキーボード表示で大きく崩れない
- [ ] 下部safe-areaに文字やボタンが隠れない
- [ ] 長文・長い単語が画面外にはみ出さない
- [ ] モーダルが画面内でスクロールできる

## 8. 注意文・App Store向け確認

- [ ] 「ご利用について」から注意文を確認できる
- [ ] 医療・診断・治療・緊急対応を目的にしていない旨が分かる
- [ ] mock AIであることをREADME上で把握できる
- [ ] secret / API key / token がコードに含まれていない
- [ ] Firebase / Gemini / 認証 / 課金に未接続であることが明記されている

## 9. Vercel Preview確認

- [ ] Vercel Preview URLが開ける
- [ ] iPhone Safariで表示できる
- [ ] PC幅でも表示できる
- [ ] ブラウザリロード後も履歴が残る
- [ ] VercelのBuild Logsにエラーがない

## 10. 次Phaseへ進む条件

- [ ] CIが成功している
- [ ] Vercel Previewが表示できる
- [ ] スマホ操作で明らかな崩れがない
- [ ] 重大なクラッシュがない
- [ ] mock版として一通り触れる

ここまで確認できたら、Phase 2でGemini実接続準備へ進めます。
