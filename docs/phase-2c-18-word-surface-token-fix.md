# Phase 2C-18: Word Surface Token Fix

## 目的

言葉の水面で、長いフレーズが枠外へ突き抜ける問題を修正する。

水面に表示するものは、長い文や句ではなく、短い単語の粒にする。

## 方針

- 全部表示しようとしない
- 2〜6文字の短い語だけを表示する
- 長いフレーズは分解する
- 表示語数は最大8個にする
- 枠内に収まらない語は表示しない
- 他の語と重なる場合は再配置する
- 置けない語は諦める

## 実装

- `keywordField.ts`
  - `IMPORTANT_COMPOUNDS` を追加
  - 長い日本語チャンクから重要語だけを抽出
  - 7文字以上の候補は基本的に除外
  - 表示候補を最大8語に制限
  - 水面枠外判定 `isInsideSurface` を追加
  - 重なり判定の余白を追加
- `phase-2c-17-fixes.css`
  - 横組み固定
  - バブル最大幅をさらに縮小
  - はみ出しは ellipsis で保護

## 期待される見え方

- 大きい語は1〜2個
- 中くらいの語が数個
- 互いに余白を持つ
- 水面の枠外に出ない
- 長いフレーズ看板にならない
