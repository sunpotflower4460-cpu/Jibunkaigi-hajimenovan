# OTHERS Reactions

## Purpose

OTHERS shows the other mirror angles that did not speak as the main reply.

## Rules

- Do not show OTHERS for 心の鏡.
- For a normal agent reply, show exactly four other agents.
- Do not include the selected agent.
- Keep each reaction short.
- Keep each reaction aligned with the agent mirror angle.

## Expected set

- If レイ speaks: ジョー, ケン, ミナ, サトウ.
- If ジョー speaks: レイ, ケン, ミナ, サトウ.
- If ケン speaks: レイ, ジョー, ミナ, サトウ.
- If ミナ speaks: レイ, ジョー, ケン, サトウ.
- If サトウ speaks: レイ, ジョー, ケン, ミナ.
- If 心の鏡 speaks: none.

## Implementation

- `src/services/othersReactions.ts` owns OTHERS generation and normalization.
- `src/services/ai.ts` delegates mock OTHERS generation to that service.
- `src/services/storage.ts` normalizes loaded/saved reactions so older data is repaired.

## Future Gemini integration

When OTHERS are generated by Gemini, never save raw model output directly. Normalize the result before saving.
