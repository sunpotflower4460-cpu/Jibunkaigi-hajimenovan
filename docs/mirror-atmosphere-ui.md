# Mirror Atmosphere UI

## 目的

じぶん会議の世界観を「本当の自分を写す鏡」として感じられるように、背景へ薄い水面・イルカ・さざ波・光粒を追加する。

これはメイン機能ではなく、内省の空気を整えるためのレイヤーである。

## 方針

- イルカはキャラクターではなく、心の水面を泳ぐ影
- 2匹のイルカは、意識と無意識、問いと答え、沈む自分と浮かぶ自分の象徴
- さざ波は、入力や反応で内側が揺れる感覚の比喩
- 妖精は前面に出さず、小さな光粒として扱う
- UI操作や文字の読みやすさを邪魔しない
- 動きすぎない
- `prefers-reduced-motion` を尊重する

## 現在の実装

- `src/components/MirrorAtmosphere.tsx`
  - 2匹のイルカのシルエット
  - 2つの波紋
  - 3つの小さな光粒
- `src/index.css`
  - イルカのゆっくりした泳ぎ
  - さざ波の呼吸
  - 光粒のゆるい漂い
  - スマホでは少し薄くする

## やらないこと

- イルカを前面キャラクター化しない
- 妖精を会話キャラクター化しない
- 派手な演出にしない
- 画面遷移やボタン操作を邪魔しない
- 重いCanvas/WebGL表現にしない

## 今後

ただし、まずは背景レイヤーとして十分に控えめな状態を確認する。
