# Phase 2C-17: Start / Agent / Word Surface Fix

## 目的

実機確認で見えた以下の違和感を修正する。

- 開始前に大きい注意カードが出て、雰囲気を壊している
- 生成AI接続に関する注釈が制作途中感を出している
- エージェント説明の主文を、より本質的な鏡表現へ寄せたい
- 言葉の水面で語同士が重なり、読みづらい

## 方針

- 注意事項は入口で大きく出しすぎず、利用規約・プライバシーポリシー・ご利用についてから開ける形にする
- 本実装予定の注釈は開始画面から出さない
- エージェントの主説明を以下へ変更する
  - レイ: 言葉にならない声を映す鏡
  - ジョー: 残っている情熱を映す鏡
  - ケン: 構造と前提を映す鏡
  - ミナ: 感情と安心を映す鏡
  - サトウ: 盲点と現実の壁を映す鏡
- 言葉の水面は表示数を絞り、衝突回避を入れる

## 実装

- `IntroLegalSoftener`
  - 開始画面に出る大きな注意/版情報カードをやわらかく非表示にする
- `agents.ts`
  - エージェント主説明を差し替え
- `keywordField.ts`
  - 表示語数を12個に抑制
  - 最大文字サイズを抑制
  - 配置時に矩形衝突判定を追加
  - 重なる場合は再試行し、配置できない語は表示しない
- `phase-2c-17-fixes.css`
  - 非表示カード用CSS
  - 言葉の水面の横組み/nowrapを補正
