# Phase 3H: Mirror UX Polish

## 目的

じぶん会議のUI/UXを、より「心を映す鏡」として感じられる方向へブラッシュアップする。

## 方針

- App Store申請前なので、大規模なUI改造は避ける
- 既存の会話UIや保存処理を壊さない
- チャットアプリ感を少し弱め、鏡の部屋・心の水面としての雰囲気を足す
- クリックや入力を邪魔しない装飾に留める
- モーダル表示中は邪魔にならないように消す

## 変更内容

- `MirrorUxPolish` コンポーネントを追加
  - 利用規約同意後にだけ表示
  - 画面上部に小さな内省キューを表示
  - 「言葉を置く」→「5つの鏡が映す」→「最後は自分に返る」の流れを静かに示す
- `mirror-ux-polish.css` を追加
  - 心の水面の中心に薄い呼吸する円を表示
  - UI操作を邪魔しない `pointer-events: none`
  - モーダル表示中は非表示
  - reduced motion に対応
- `main.tsx` に `MirrorUxPolish` を常駐

## 期待する体験

- 初回後の画面が、ただのAIチャットではなく「鏡の部屋」に近づく
- ユーザーが「答えをもらう」より「自分の言葉を置く」感覚を持ちやすくなる
- 5人のエージェントが、キャラクターではなく鏡の角度として見えやすくなる

## 手動確認

- 初回同意前には表示されない
- 同意後、画面上部に小さな鏡キューが表示される
- キューが入力やボタン操作を邪魔しない
- モーダル表示中にキューが消える
- スマホで文字や円がうるさすぎない
- reduced motion環境で過剰に動かない

## 保留したこと

`AppStable.tsx` の大きな文言差し替えは、競合リスクがあるため今回は見送った。必要なら次Phaseで、より小さい差分に分けて文言だけを調整する。
