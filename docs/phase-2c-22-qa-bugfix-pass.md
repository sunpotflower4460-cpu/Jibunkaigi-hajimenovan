# Phase 2C-22: QA Bugfix Pass

## 目的

実機QAで出やすい、細かいが体験を壊す不具合を先回りして修正する。

## 修正内容

- Dive Tools のスマホアクセシビリティ補強
  - 右下メニューの最大幅・safe-area調整
  - パネル内スクロールの安定化
  - 旧フローティングボタンの非表示維持
- 別の Dive Tool を開く前に、既存の開いているdialogを閉じる
  - `closeDivePanels()` で close-all イベントを出す
  - 既存dialog内の「閉じる」ボタンも押して、旧実装パネルも閉じる
  - 開閉タイミングを少し遅らせ、モーダル重なりを防ぐ
- 設定パネルの改善
  - close-allイベントに対応
  - Escキーで閉じる
  - 「今後ここに追加する候補」のような開発途中感のある枠を削除

## 期待される効果

- 設定を開いたまま言葉の水面を開くなどの操作で、モーダルが重なりにくくなる
- 右下の潜るメニューがスマホで扱いやすくなる
- リリース前の見え方として制作途中感が減る

## 手動確認

- 潜る → 設定 → 潜る → 言葉 の順に押して、設定が残らない
- 潜る → 会議録 → 潜る → 付箋 の順に押して、会議録が残らない
- 設定パネルでEscキーを押すと閉じる
- スマホで潜るメニューが画面外にはみ出さない
